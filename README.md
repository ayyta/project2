# Web Development Project 2 - Guess the Flag

Submitted by: Alvis Yong

This web app: Following a flashcard like functionality, Guess the Flag presents a picture of a flag and will reveal what country it is when clicked on. Users can also go back to previous cards and go to new ones.

Time spent: 5 hours spent in total
